Программа CLI для отслеживаня доходов/расходов, в качестве тестового задания.

Для запуска у себя нужно выполнить следующие действия:

1. git clone https://notabug.org/borrya_/financial_wallet

2. Перейти в папку проекта

3. И выполнить команду python3 main.py

Описание функционала:

Точка входа - main.py, с этого места начинается весь код.
В папке db хранятся БД в формате json и она имеет начальные значения для отображения.
В папке utils храняться модули, которые хранят логически соответствующий им код, это еще называют сервисами.
Каждый модуль из папки utils содержит код, тип которого относится к названию этого же модуля. В каждом модуле имеется основной класс, описание которого содержится в комментарии.
